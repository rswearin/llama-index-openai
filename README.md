# llama-index-openai
A very simple script to perform R.A.G locally using llama index and openai. 


TO USE:

Copy main.py
Create venv and install deps

```
--requirements.txt--
python-dotenv
openai
llama-index

> pip install -r requirements.txt

```
Create .env and set OPENAI_API_KEY

Create folder './data'
Put any RAG data inside

> python3 main.py
